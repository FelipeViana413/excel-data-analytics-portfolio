Repositório com projetos práticos de análise e automação de dados
usando Excel avançado, Power Query, VBA e Power BI. Desenvolvido
como portfólio de transição para a área de dados/analytics.

---

### 1. Power Query — ETL e transformação de dados
`/01-power-query`

Pipeline de ingestão e limpeza de dados de vendas a partir de
múltiplas fontes (CSV, planilhas, banco de dados). Inclui:
- Consolidação de bases com diferentes estruturas
- Tratamento de nulos, duplicatas e tipos de dados
- Atualização automática com um clique

### 2. VBA & Macros — Automação de relatórios
`/02-vba-macros`

Automação completa do processo de geração de relatórios mensais,
reduzindo o tempo de execução de 2h para 5 minutos. Inclui:
- Leitura e consolidação de arquivos externos
- Formatação condicional automatizada
- Envio de e-mail com relatório anexado

### 3. Dashboards — Visualização para tomada de decisão
`/03-dashboards-bi`

Painéis interativos com segmentações, KPIs e gráficos dinâmicos
conectados a bases atualizáveis. Inclui:
- Dashboard de performance comercial
- Análise de variação mês a mês (MoM)
- Indicadores com alertas visuais por meta

### 4. Fórmulas avançadas
`/04-formulas-avancadas`

Aplicações práticas de fórmulas dinâmicas modernas:
- LAMBDA, LET e funções personalizadas
- SEQUENCE e FILTER para relatórios automáticos
- Fórmulas de array para análises complexas
